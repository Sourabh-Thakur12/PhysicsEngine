# C++ Physics Engine (From Scratch)

## Overview
This project is a custom-built physics engine developed entirely from scratch in C++. The goal is to deeply understand motion dynamics, force integration, and collision systems without relying on existing game engines.

## Objective
To implement a deterministic motion simulation system using core mathematical and physics principles.
